# Магазин продуктов API

API для интернет-магазина продуктов с корзиной, категориями и товарами.

## Функционал

### Категории и подкатегории
- Управление категориями и подкатегориями через админку (создание, редактирование, удаление)
- Категории и подкатегории имеют: наименование, slug-имя, изображение
- Подкатегории связаны с родительской категорией
- Эндпоинт просмотра всех категорий с подкатегориями (с пагинацией)

### Продукты
- Управление продуктами через админку (добавление, изменение, удаление)
- Продукты относятся к подкатегории (и соответственно к категории)
- Продукты имеют: наименование, slug-имя, изображение в 3-х размерах (small, medium, large), цену
- Эндпоинт вывода продуктов с пагинацией

### Корзина
- Добавление товара в корзину
- Изменение количества товара в корзине
- Удаление товара из корзины
- Полная очистка корзины
- Просмотр состава корзины с подсчетом количества товаров и суммы
- Операции с корзиной доступны только авторизованным пользователям (только со своей корзиной)

### Авторизация
- Аутентификация по токену (Token Authentication)

### Документация
- Swagger UI: `/swagger/`
- ReDoc: `/redoc/`

### Тесты
- Покрыты автотестами GET и POST методы

### Фикстуры
- Начальные данные загружаются командой `load_fixtures`

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <repo-url>
   cd grocery_store
   ```

2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate     # Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Примените миграции:
   ```bash
   python manage.py migrate
   ```

5. Создайте суперпользователя для доступа к админке:
   ```bash
   python manage.py createsuperuser
   ```

6. Загрузите фикстуры (опционально):
   ```bash
   python manage.py load_fixtures
   ```

7. Запустите сервер разработки:
   ```bash
   python manage.py runserver
   ```

## API Endpoints

### Публичные эндпоинты (доступны всем)

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/categories/` | Список категорий с подкатегориями (пагинация) |
| GET | `/api/products/` | Список продуктов (пагинация) |

### Эндпоинты корзины (только для авторизованных)

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/api/cart/` | Получить состав корзины |
| POST | `/api/cart/` | Добавить товар в корзину (`{"product": id, "quantity": count}`) |
| PATCH | `/api/cart/` | Изменить количество товара (`{"product": id, "quantity": count}`) |
| DELETE | `/api/cart/` | Удалить товар (`{"product": id}`) или очистить корзину (без параметров) |

### Авторизация

| Метод | URL | Описание |
|-------|-----|----------|
| POST | `/api/api-token-auth/` | Получить токен (`{"username": "...", "password": "..."}`) |

## Swagger документация

После запуска сервера документация доступна по адресам:

- **Swagger UI**: http://localhost:8000/swagger/
- **ReDoc**: http://localhost:8000/redoc/

## Админка

Доступна по адресу: http://localhost:8000/admin/

В админке можно управлять:
- Категориями (с встроенным управлением подкатегориями)
- Подкатегориями
- Продуктами
- Корзинами пользователей

## Тестирование

Запуск тестов:
```bash
python manage.py test
```
